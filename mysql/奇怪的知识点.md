# 如何查询 比较NULL值？
众所周知 任何带null的运算结果都为NULL 那么要进行带有NULL的比较和查询就需要  
1. 安全等于 <=>  此符号如果左右均为null 那么返回1 如果有一个为null 那么返回 0
2. IS NULL / IS NOT NULL
3. 函数 ISNULL()
